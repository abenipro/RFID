Description About The Project

This is a full integrated remote alarming system that monitor students and send text message to the parents. It consists three monitoring systems which are.

1. system use at the gate of the school

2. system use in the class

3. system use in the home

Using RFID reader or RX & TX and programing codes we can monitor your children at school from far distance and alarm the parents every time when the status of the student change from predetermined status.

We provide the following product and sevices

1. Full product

2. Product with onsite instalment

3. If there is problem on the system we provide 1 year guaranty

4. Modification of code of the system and hardware accessories

GSM is very easily accessible and cheap network and as services are being built on top of this technology, the number of interested parties will increase significantly .

Using the same basic principle the concept of this project can also be extended to many application and scenarios, to mention some of them; home, car and property security, home arrest, physical impaired people, elders and also we can use the principle for house arrest. Vehicle tracking, keeping prisoners from escaping and many real time applications can also be done by using this concept.
